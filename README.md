# Data Distribution Polices
## Kentucky Aerial Photography & Elevation Data Program 

Draft June 2026

### Overview

Kentucky's Aerial Photography and Elevation Data (KYAPED) Program, also known as KyFromAbove, provides geospatial data for broad range of applications. This document outlines the Data Distribution Policy associated with the KYAPED Program. 

### License

Data acquired through the KYAPED effort and hosted on the AWS Registry of Open Data @ [registry.opendata.aws/kyfromabove](https://registry.opendata.aws/kyfromabove) is made available under the Creative Commons CC0 1.0 Universal (CC0 1.0) Public Domain Dedication.

### Methods of data distribution 

Distribution is limited to any cloud-based download services available on the AWS platform. There is no cost to download data from the AWS Registry of Open Data.

*If requested, KYAPED Program Partners (those that have a signed MOA with the Commonwealth) shall receive one (1) copy of the data on a portable hard drive. If a DGI portable hard drive is used for data transfer it must be returned within a reasonable timeframe (1-2 weeks).

*All data products are tiled and named using the Kentucky 5K tiling grid and provided in Kentucky Single State Plane Zone (FIPS: 1600) (EPSG: 3089). Data products will not be reprojected, renamed, mosaicked, or converted to different file format.

### File Formats

Below is a full listing of the data products that will be available via cloud-based download services.

#### Primary Data Products

- Aerial photography 4-band (RGB/NIR) cloud-optimized GeoTIFFs with TFW world files

- Aerial oblique (RGB) cloud-optimized GeoTIFFs

- Digital Elevation Models (DEMs) in cloud-optimized GeoTIFF format with TFW world files

- Lidar Point Cloud in LAZ (compressed LAS) and COPC (Cloud Optimized Point Cloud) formats

#### Derivative Data Products

- Vector contours and spot elevations in GeoPackage format
- Topographic Map Series for latest elevation data in cloud-optimized and plain GeoTIFF, Geospatial PDF, and full-collar PNG
